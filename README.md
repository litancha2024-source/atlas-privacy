# Atlas OS Privacy Policy

Static privacy policy website for Atlas OS, built with HTML and CSS for GitHub Pages.

## Files

- `index.html` — privacy policy content
- `styles.css` — responsive page styling
- `.nojekyll` — serves the site directly without Jekyll processing

## Local Preview

Open `index.html` in a browser, or run a local static server from this directory:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Publish with GitHub Pages

1. Create or open the GitHub repository `litancha2024/atlas-privacy`.
2. Push this project to the repository's `main` branch.
3. Open the repository on GitHub.
4. Select **Settings → Pages**.
5. Under **Build and deployment**, select **Deploy from a branch**.
6. Select the `main` branch and the `/(root)` folder.
7. Select **Save** and wait for GitHub Pages to finish publishing.

The published privacy policy URL will be:

```text
https://litancha2024.github.io/atlas-privacy/
```

## Push Commands

After authenticating GitHub CLI or Git on this Mac:

```bash
git push -u origin main
```
